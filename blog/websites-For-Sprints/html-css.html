Positioning in CSS

Relative Positioning

Relative positioning is a CSS positioning property that positions an element relative to its normal position. When you apply relative positioning to an element, it will still take up space in its original position, but you can move it from that position using the top, bottom, left, and right properties.

Absolute Positioning

Absolute positioning is a CSS positioning property that positions an element relative to its nearest positioned ancestor element. If an element does not have a positioned ancestor, it will be positioned relative to the document body. When you apply absolute positioning to an element, it will not take up space in its original position, and you can move it anywhere on the page using the top, bottom, left, and right properties.

Fixed Positioning

Fixed positioning is a CSS positioning property that positions an element relative to the browser window, rather than the document or its positioned ancestors. When you apply fixed positioning to an element, it will not move when the user scrolls the page, and you can position it anywhere on the screen using the top, bottom, left, and right properties.
